Youtube Channel Slider displays your youtube channel using slider (show thumbs).

== Description ==

Youtube Channel Slider displays your youtube channel videos either with excerpt or thumbnail image using slider.
You can customize the slider in many ways (width, height, video per slide, no of videos to display & more).

It extract the thumbnail from videos in the youtube channel.


If you have a feature that you would like to get added in future versions, feel free to contact me at http://www.kesweh.com/contacto/.

If you find it useful please don't forget to rate this plugin.

== Installation ==

= Installation =

1. You can use the built-in installer.
     OR         
     Download the zip file and extract the contents.
     Upload the 'youtube-channel-slider' folder to your plugins directory (wp-content/plugins/).
1. Activate the plugin through the 'Plugins' menu in WordPress.

Now go to **Settings** and then **Youtube Channel Slider** to configure any options as desired.

= How to use =

In order to display the youtube video channel slider, you have three options

1. Simply place `<?php if (function_exists('ycs_show')) echo ycs_show(); ?>` in your theme. 
1. Add the shortcode '[ycs]'.
1. Using widget.
